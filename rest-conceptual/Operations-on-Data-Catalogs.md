The Data Catalogs operations can only be called using a user account that has **access to create resources in an Azure subscription**. To authenticate to use these APIs, you must follow the standard procedure for all Azure Resource Manager requests, see [Authenticating Azure Resource Manager requests](https://msdn.microsoft.com/library/azure/dn790557.aspx).
